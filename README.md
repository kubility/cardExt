# 民间卡牌扩展包

## 介绍

民间卡牌扩展包，包含多种自定义卡牌，为无名杀游戏添加更多玩法和策略选择。

## 版本信息

- **版本**：1.0.0
- **作者**：搬山改
- **类型**：卡牌扩展

## 功能特性

### 配置选项

扩展包提供以下配置选项，可在游戏设置中启用或禁用：

1. **民间基本** - 启用自定义基本牌
2. **民间锦囊** - 启用自定义锦囊牌
3. **民间武器** - 启用自定义武器牌
4. **民间装备** - 启用自定义装备牌
5. **场景卡牌** - 启用场景卡牌

## 安装说明

1. 将整个`民间卡牌`文件夹放入无名杀的`extension`目录下
2. 启动无名杀游戏
3. 在游戏设置中启用所需的卡牌类型
4. 开始游戏即可使用新增的卡牌

## 技术说明

### 代码规范

- 使用ES6+语法
- 遵循无名杀v1.10+引擎规范
- 所有`content`和`cost`函数使用`async/await`
- 标准函数签名：`async function(event, trigger, player)`
- 异步交互处理：使用`.forResult()`获取结果

### AI配置

- 所有主动技能必须提供`ai`对象
- 包含`order`和`result`配置
- 根据技能效果合理配置AI标签

## 注意事项

1. 本扩展包需要无名杀v1.10+版本
2. 某些卡牌效果可能与其他扩展包产生交互
3. 建议在熟悉基本规则后再使用自定义卡牌
4. 平衡性调整请参考游戏实际体验
